SuperStore Sales Data Pipeline & Dashboard using Snowflake and Power BI

Built an end-to-end data analytics project using Snowflake and Power BI based on SuperStore sales data. Designed a medallion-style pipeline by loading raw CSV data into a Bronze table using Snowflake Stage and File Format objects. Performed data cleansing, null handling, date conversion, and datatype transformations using SQL. Created dimension tables (Customer, Product, Location) and a Fact Orders table using star schema modeling. Applied deduplication and aggregation logic for better data quality. Connected transformed data to Power BI and developed interactive dashboards for sales, profit, customer segments, product categories, and regional performance insights.

Skills : Snowflake, SQL, Power BI, Data Modeling, ETL
